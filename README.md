## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/stacoit.git
   cd staco-data-solutions
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```

4. **Open browser**
   Visit [http://localhost:3000](http://localhost:3000)
